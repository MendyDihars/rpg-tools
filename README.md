# RPG Tools

This project is a single-page application scaffolded with Vite and React.

## Requirements

- Node.js 18+ (20.10.0 is fine)
- Yarn 1.22+

## Scripts

- `yarn install` – install dependencies
- `yarn dev` – start the development server with hot module replacement
- `yarn build` – create a production-ready build under `dist`
- `yarn preview` – locally preview the build output

## Static Hosting

Run `yarn build` and deploy the generated `dist` directory to your static hosting provider.

## Styling

Tailwind CSS 4 powers the styling pipeline. Customize design tokens in `tailwind.config.js` and extend global styles via `@layer` directives in `src/index.css`.
